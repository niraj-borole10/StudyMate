Application Setup & Instructions

1. Google Cloud Configuration

StudyMate interacts with Gmail, Calendar, and Drive. Make sure you place your Google Developer client credentials file named client_secret.json in your project root: C:\Users\Ulhas\Desktop\MCP\client_secret.json.

Redirect URI Required: http://localhost:5000/api/auth/callback

2. MongoDB Connection Link

By default, the application connects to a local MongoDB instance. If you want to use your own database link (such as MongoDB Atlas), edit the MONGO_URI environment variable inside the .env file in your project root:

MONGO_URI=mongodb+srv://<username>:<password>@cluster0.example.mongodb.net/studymate

3. Gemini AI Extractions

Ensure you have set the GEMINI_API_KEY environment variable in your .env file to enable intelligent homework metadata parsing and study schedule optimization.
